图片标签

属性 属性值 说明
src 图片路径 必须属性
alt 文本 替换文本。图像不能显示的文字
title 文本 提示文本。鼠标放到图像上，显示的文字
width 像素 设置图像的宽度
height 像素 设置图像的高度
border 像素 设置图像的边框粗细（一般在CSS设置）

图片路径

相对路径

相对路径分类 符号 说明
同一级路径 图像文件位于HTML文件同一级 如
下一级路径 / 图像文件位于HTML文件下一级 如
上一级路径 ../ 图像文件位于HTML文件上一级 如

绝对路径

例：”D:\web\images\logo.gif”或完整的网络地址“http://www.itcast.cn/images/logo.gif”。

超链接标签⭐

属性 作用
href 用于指定链接目标的url地址，（必须属性）当为标签应用href属性时，它就具有了超链接的功能
target 用于指定链接页面的打开方式，其中_self为默认值（当前窗口）， _blank为在新窗口中打开方式

当href是一个文件或压缩包时，点击就会下载该文件。
